Step 1 — Create your Airtable base

1
Go to airtable.com and sign in
2
Click + Create → Start from scratch → name it Parent Comms
3
Rename the default table to Students and add these fields:
NameSingle line text
Mom NameSingle line text
Mom EmailEmail
Mom PhonePhone number
Dad NameSingle line text
Dad EmailEmail
Dad PhonePhone number

Step 2 — Create the Communication Logs table

1
Click + Add table and name it Communication Logs
2
Add these fields:
Student NameSingle line text
Log DateDate
MethodSingle select *
NotesLong text
Next ContactDate
* For Method, add these options to the single select field:
Phone Call · Email · Voicemail Left · PPT Email · PPT Meeting · No Response · Parent Reached Out · Other

Get your Airtable API token

1
Go to airtable.com/create/tokens
2
Click Create new token
3
Give it a name (e.g. "Parent Comms"), set scope to data.records:read and data.records:write, and add your Parent Comms base
4
Copy the token and paste it below — it starts with pat

Get your Base ID

1
Open your Parent Comms base in Airtable
2
Look at the URL — it looks like airtable.com/appXXXXXXXXXX/...
3
Copy the part that starts with app and paste it below
Please enter both your API token and Base ID.
